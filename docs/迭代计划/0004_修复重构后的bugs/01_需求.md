### 修复源代码中的bug

Empyrical类进行重构之后，出现了一些bug,需要修复。

### 实施步骤
0. pip install -U .   安装fincore
1. 运行pytest tests -n 10，查看测试结果
2. 寻找错误最多的错误类型，优先修复这些错误
3. pip install -U .   安装fincore
4. 修复错误后，再次运行pytest tests -n 10，查看测试结果
5. 如果还有错误，重复步骤2和步骤3，直到所有错误都修复

### 限制

1. 函数和方法返回结果是具体的值，或者Series,或者DataFrame，不是numpy的array,如果是numpy的array,需要转换为Series或者DataFrame
2. 出现错误的时候，优先查看_empyrical中的相应的函数是如何实现的，原先的函数都能通过测试，逻辑是正确的。
3. 不允许修改测试用例中的输入和期望值。
