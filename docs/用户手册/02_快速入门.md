# 快速入门

5 分钟上手 fincore 核心功能。

## 准备数据

fincore 的核心输入是**日收益率序列** (`pd.Series`)：

```python
import numpy as np
import pandas as pd

np.random.seed(42)
dates = pd.date_range("2020-01-01", periods=500, freq="B", tz="UTC")
returns = pd.Series(np.random.normal(0.0005, 0.015, 500), index=dates, name="strategy")
benchmark = pd.Series(np.random.normal(0.0003, 0.012, 500), index=dates, name="benchmark")
```

## 方式一：Flat API

最简洁的调用方式，一个函数计算一个指标：

```python
import fincore

print(fincore.sharpe_ratio(returns))      # 夏普比率
print(fincore.max_drawdown(returns))      # 最大回撤
print(fincore.annual_return(returns))     # 年化收益
print(fincore.annual_volatility(returns)) # 年化波动率
```

## 方式二：Empyrical 类

100+ 个指标函数，覆盖面最广：

```python
from fincore import Empyrical

# 基础指标
stats = Empyrical.perf_stats(returns)
print(stats)

# 回撤分析
dd_table = Empyrical.gen_drawdown_table(returns, top=5)

# 滚动指标
rolling_sr = Empyrical.rolling_sharpe(returns, rolling_window=63)

# 按年聚合
yearly = Empyrical.annual_return_by_year(returns)
```

## 方式三：Empyrical 实例

绑定数据后自动填充参数：

```python
emp = Empyrical(returns=returns, factor_returns=benchmark)

# 不需要传 returns，自动使用实例绑定的数据
print(emp.win_rate())
print(emp.max_drawdown_days())
print(emp.treynor_ratio())  # 自动使用 factor_returns
```

## 方式四：AnalysisContext

适合需要计算大量指标的场景，所有结果自动缓存：

```python
from fincore import analyze

ctx = analyze(returns, factor_returns=benchmark)

# cached_property: 首次计算后缓存
print(ctx.sharpe_ratio)
print(ctx.max_drawdown)
print(ctx.alpha)

# 导出
print(ctx.perf_stats())
print(ctx.to_json())
```

## 方式五：策略报告

一行代码生成 HTML/PDF 报告：

```python
fincore.create_strategy_report(
    returns,
    title="My Strategy",
    output="report.html",
)
```
