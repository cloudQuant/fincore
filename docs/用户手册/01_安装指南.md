# 安装指南

## 系统要求

- Python 3.9+
- 支持 macOS、Linux、Windows

## 安装方式

### 开发模式（推荐）

从源码安装，适合开发和调试：

```bash
git clone <repo-url> fincore
cd fincore
pip install -e ".[dev]"
```

### 最小安装

仅安装运行时依赖：

```bash
pip install .
```

### 依赖说明

**核心依赖**（自动安装）：

| 包 | 用途 |
|---|------|
| `numpy` | 数值计算 |
| `pandas` | 数据结构（Series, DataFrame） |
| `scipy` | 统计分布、优化算法 |

**可选依赖**：

| 包 | 用途 | 安装方式 |
|---|------|---------|
| `matplotlib` | 图表绘制、tear sheet | `pip install matplotlib` |
| `statsmodels` | GARCH 模型、回归分析 | `pip install statsmodels` |
| `scikit-learn` | 风格分析、回归归因 | `pip install scikit-learn` |

**开发依赖**：

```bash
pip install -e ".[dev]"
# 包含: pytest, pytest-cov, pytest-xdist, ruff, mypy
```

## 验证安装

```python
import fincore
print(fincore.__version__)

# 快速测试
import numpy as np
import pandas as pd

returns = pd.Series(np.random.normal(0, 0.01, 252))
print(f"Sharpe: {fincore.sharpe_ratio(returns):.4f}")
```

## 运行测试

```bash
# 快速测试
python -m pytest tests -q

# 并行测试
python -m pytest tests -n 8 -q

# 带覆盖率
python -m pytest tests --cov=fincore --cov-report=term-missing
```
